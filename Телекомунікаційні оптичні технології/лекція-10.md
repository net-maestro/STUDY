# Лекція 10. Мережі FDDI-2 (Топології, OBS, Кабельна система)

## 10.1. Топології мереж FDDI

### Логічна та фізична топологія
*   **Логічна топологія:** Завжди є **кільцем** (шлях руху даних).
*   **Фізична топологія:** Може бути різноманітною (розміщення пристроїв та кабелів).
*   **Базове з'єднання:** "Точка-точка" (дозволяє використовувати різні типи кабелів на різних ділянках).

### Основні варіанти фізичної топології

| Тип топології | Опис | Особливості та застосування |
| :--- | :--- | :--- |
| **1. Точка-точка** | З'єднання двох станцій типу **SAS**. | Найпростіша топологія. Може використовувати ВОК або кручену пару. |
| **2. Подвійне кільце** | З'єднання станцій типу **DAS** безпосередньо в кільце. | • Кожна станція — критичний вузол.<br>• Відключення станції → режим **WRAP** (згортання).<br>• Використовується для невеликої кількості станцій. |
| **3. Окремий концентратор** | Використовується один концентратор типу **NAC**. | • Концентратор має внутрішню шину.<br>• До М-портів підключаються SAS або DAS.<br>• Можливе підключення **Dual Homing**. |
| **4. Дерево концентраторів** | Ієрархічна зірка з кореневим концентратором (**NAC**). | • Гнучкість додавання/видалення станцій без розриву мережі.<br>• Зручно для однієї будівлі/поверху.<br>• Концентратор автоматично відключає несправні станції. |
| **5. Подвійне кільце дерев** | Дерева концентраторів, корені яких (**DAC**) увімкнені в подвійне кільце. | • **Найуніверсальніша та гнучка топологія**.<br>• Поєднує надійність кільця та гнучкість дерева.<br>• Зберігає цілісність при пошкодженні ліній або відключенні вузлів. |

### Механізм Dual Homing
*   **Призначення:** Підвищення надійності підключення окремої станції (DAS) або концентратора (DAC).
*   **Принцип:** Пристрій підключається своїми портами **A і B** до **М-портів** різних концентраторів мережі.
*   **Режим роботи:**
    *   Один канал активний (робочий).
    *   Другий канал у стані очікування (резервний).
    *   При розриві робочого сегмента — автоматичне перемикання на резервний.
*   **Застосування:** Банки, фінансові установи, об'єкти з високими вимогами до безперервності зв'язку.

## 10.2. Оптичний обхідний перемикач (OBS)

### Призначення та принцип дії
**OBS (Optical Bypass Switch)** — пристрій для збереження цілісності подвійного кільця при відключенні живлення станції **DAS** або концентратора **DAC**.

*   **Живлення:** Отримує від станції (5 В).
*   **Робочий стан (живлення є):** Сignal проходить через станцію.
*   **Байпасний стан (живлення немає):** Механічно замикає вхідні та вихідні волокна, обходячи станцію.

### Процес реконфігурації при аварії
1.  **Виключення вузла:** OBS переходить у байпасний режим (замикання кільця через себе).
2.  **Реініціалізація:** Маркерне кільце ініціалізується заново (час: **10–150 мс**).

> **Обмеження OBS:**
> *   Максимальна кількість послідовно вмиканих OBS у кільці: **4 штуки**.
> *   Причина: Накопичення оптичних втрат (загасання).
> *   Допустиме загасання на один OBS: до **2.5 дБ**.
> *   Рекомендований запас потужності: **2–3 дБ**.

### Розрахунок втрат (Приклад)
| Параметр | Початкові втрати (дБ) | Після експлуатації (дБ) |
| :--- | :---: | :---: |
| Загасання в кабельній системі | 3.0 | 3.5 |
| Втрати в з'єднувачах | 1.2 | 2.8 |
| Втрати OBS | 1.1 | 2.1 |
| **Загальні втрати** | **5.3** | **8.4** |
| **Запас (Бюджет 11 дБ)** | **4.7** | **2.6** |

### Типи OBS
| Тип | Особливості | Вартість |
| :--- | :--- | :--- |
| **Багатомодовий** | Простіший у виготовленні, менші втрати | Нижча |
| **Одномодовий** | Складніше через малий діаметр серцевини, вищі втрати | У 3–4 рази вища |

**Характеристики одномодового OBS:**
*   Довжина хвилі: 1280–1650 нм.
*   Внесені втрати: 1.3 дБ (тип.), 2.5 дБ (макс.).
*   Час спрацьовування: 50 мс.

## 10.3. Кабельна система та рівень PMD

### Стандарти фізичного рівня (PMD)
Рівень PMD стандартизує фізичну передачу даних (кабель, роз'єми, оптичні параметри).

| Стандарт | Середовище | Особливості |
| :--- | :--- | :--- |
| **MMF-PMD** | Багатомодове волокно (62.5/125 мкм) | Базовий стандарт. Довжина хвилі 1300 нм. |
| **SMF-PMD** | Одномодове волокно | Для великих відстаней. Ідентичні вимоги до потужності приймача. |
| **TP-PMD** | Вита пара (мідь) | UTP Cat.5 (4 пари) або STP Type 1/2 (2 пари). Макс. відстань 100 м. |

### Оптичні характеристики (MMF/SMF)
*   **Довжина хвилі:** 1300 нм.
*   **Мін. потужність на вході приймача:** -31 дБм (BER ≤ 2.5·10⁻¹⁰).
*   **Макс. потужність передавача:** -14 дБм.
*   **Мін. потужність передавача:** -20 дБм.
*   **Макс. втрати між станціями:** 11 дБ.
*   **Макс. втрати на км кабелю:** 2.5 дБ.

### Функція реєстрації сигналу (Signal Detect)
Контроль цілісності лінії зв'язку на рівні PMD.

*   **Оптичне середовище:**
    *   Signal_Detect = **ON**, якщо потужність > **-43.5 дБм**.
    *   Signal_Detect = **OFF**, якщо потужність < **-45 дБм**.
    *   **Гістерезис:** 1.5 дБ (для запобігання хибним перемиканням).
*   **Мідне середовище (TP-PMD):**
    *   Контроль цілісності пар через вимірювання постійного струму (петля на віддаленому кінці).

### Оптичні з'єднувачі
| Тип | Опис |
| :--- | :--- |
| **MIC** | Стандартний з'єднувач FDDI. Полярна структура, ключ для захисту від неправильного підключення. |
| **Duplex SC** | Найпоширеніший альтернативний стандарт. Компактний, дешевший, сумісний з MMF/SMF. |
| **ST, FC** | Використовуються переважно для одномодового волокна (SMF-PMD). |

### Змішана кабельна система
Мережа FDDI часто будується як комбінація різних середовищ:
1.  **Магістраль (між будинками):** Одномодове або багатомодове волокно.
2.  **Вертикальна розводка (в будівлі):** Багатомодове станційне волокно.
3.  **Горизонтальна розводка (поверх):** Оптичні шнури або вита пара (UTP Cat.5).

> **Універсальність:** Кабельна система FDDI сумісна з обладнанням Fast Ethernet, ATM та іншими змішаними рішеннями.

## Ключові терміни лекції

*   **Logical vs Physical Topology:** Логічна (кільце) проти фізичної (різноманітна) структура мережі.
*   **Dual Homing:** Механізм подвійного підключення станції до різних концентраторів для підвищення надійності.
*   **OBS (Optical Bypass Switch):** Оптичний обхідний перемикач для виключення несправної станції з кільця без розриву топології.
*   **WRAP Mode:** Режим згорнутого кільця при аварії (активується без OBS).
*   **PMD (Physical Medium Dependent):** Нижній підрівень фізичного рівня (кабель, оптику, роз'єми).
*   **MMF/SMF/TP:** Багатомодове, одномодове волокно та вита пара.
*   **Signal Detect:** Функція виявлення наявності сигналу на фізичному рівні (з гістерезисом).
*   **MIC Connector:** Стандартний оптичний роз'єм FDDI.
*   **Attenuation Budget:** Бюджет загасання (макс. 11 дБ для FDDI).
*   **NAC/DAC/SAC:** Типи концентраторів (Null/Dual/Single Attachment).

## Порівняння топологій за надійністю

| Топологія | Надійність | Гнучкість | Вартість |
| :--- | :---: | :---: | :---: |
| **Подвійне кільце (DAS)** | Середня (критичність вузлів) | Низька (реініціалізація при зміні) | Низька (без концентраторів) |
| **Дерево концентраторів** | Висока (автоматичне відключення) | Висока | Середня |
| **Подвійне кільце дерев** | **Найвища** (резервне кільце + дерева) | **Найвища** | Висока |
| **Dual Homing** | Додатковий захист для окремих вузлів | Середня | Залежить від реалізації |
